# Faunadex Arena

Projeto integrado em grupo, desenvolvido com HTML e CSS, inspirado na fauna, flora e folclore brasileiro.

O projeto começou como uma coleção de cartas digitais e evoluiu para uma arena com tabuleiro 3x3, mão do jogador e aviso de privacidade (LGPD).

**FanBoy Studios**

## Objetivo do Projeto

Desenvolver um protótipo visual de jogo de cartas digitais, unindo:

- organização em squad;
- uso de inteligência artificial para criar imagens;
- estruturação de páginas com HTML;
- estilização visual com CSS.

## Sprints

### Sprint 01 — Coleção de Cartas
Nesta etapa, foram criadas as primeiras cartas digitais do projeto:

- Curupira Guardião
- Onça-Pintada Selvagem

Também foi desenvolvida a interface inicial das cartas com HTML e CSS.

### Sprint 02 — A Planta da Arena
Nesta etapa, o projeto evoluiu para a estrutura de jogo, com:

- tabuleiro 3x3;
- mão do jogador com 5 cartas;
- reaproveitamento das cartas criadas anteriormente;
- modal de aviso de privacidade (LGPD).
