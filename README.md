# Faunadex - Coleção Brasileira

Projeto integrado desenvolvido em grupo para criar cartas digitais inspiradas na fauna, flora e folclore brasileiro.

## Tecnologias usadas

- HTML5
- CSS3
- Inteligência Artificial para geração das imagens

## Cartas criadas

- Curupira Guardião
- Onça-Pintada Selvagem

## Objetivo

Criar um protótipo visual de cartas digitais, praticando organização em squad, engenharia de prompts, estrutura HTML e estilização com CSS.
